# Lendsqr Admin Dashboard

A user management dashboard built with React, TypeScript, and SCSS.

## Quick Start

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run test` | Run tests |
| `npm run preview` | Preview production build |

## Tech Stack

- React 18 + TypeScript
- Vite
- SCSS Modules
- TanStack Query
- React Router v6
- Vitest

## Features

- Login authentication
- Users table with filtering and pagination
- User details with multiple tabs
- Blacklist/Activate user actions
- Responsive design
